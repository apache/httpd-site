Apache HTTP Server Documentation¶

The documentation is available is several formats. Downloadable formats including Windows Help format and offline-browsable html are available from our distribution mirrors. Online browsable documentation is also available:

  • Version 2.4 (Current)

  • Version 2.2 (Historical)

  • Version 2.0 (Historical)

  • Trunk (Development / Draft)

There is also a Documentation Wiki for user-contributed recipes, tips, and tricks.

The documentation is maintained by the documentation project. We'd love to have your help to improve the docs.
