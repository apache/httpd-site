Mailing list archives browser¶

mod_mbox is a mailing list archive browser. Functionality includes:

  • A dynamic browsing interface, using AJAX (Asynchronous Javascript And XML)

  • A flat, Javascript-less interface, using static XHTML

  • Mail attachments and MIME parts viewing and downloading

News¶

July and August 2005 - Summer Of Code projects ( httpd-mbox-if and httpd-mbox-search ) renew and improve the module.

July 17, 2001 - First mod_mbox release.

Subversion repository¶

Development of mod_mbox can be followed through the Subversion repositories. For public access you can use :

% svn checkout http://svn.apache.org/repos/asf/httpd/mod_mbox/trunk/
mod_mbox

History¶

mod_mbox was originally written by Justin Erenkrantz in 2001, and Paul Querna also worked on the module from time to time. More recently, Maxime Petazzoni and Russell Johnson joined the "team" as Summer Of Code students. Russell implemented the full text search in mod_mbox , while Maxime worked on improving the module's user interface.
