Just the facts ... where and how to contribute.

Where¶

The source code is in svn at:

  • Trunk: https://svn.apache.org/repos/asf/httpd/httpd/trunk
  • 2.4 (stable): https://svn.apache.org/repos/asf/httpd/httpd/2.4.x

The source is also mirrored on Github

(Further details in Developer Info)

The documentation source is in each version's source tree, under docs/manual/

Website source is at: https://svn.apache.org/repos/asf/httpd/site/trunk/content

How¶

Patches may be submitted by mailing output of svn diff to the dev@httpd mailing list, or via a pull request at the above Github repo.

Documentation patches may be sent to the docs@httpd mailing list.

(Further details about how to generate an svn diff are in the devnotes
